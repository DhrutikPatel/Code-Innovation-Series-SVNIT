Mind Install

We have made our project on Edutech.

In this project we have made a doubts solving platform like discussion forum in which students can
ask there doubts occured during online classes on different subjects to respective teacher. So that
the students doubts can be solved time effective manner. If in future another students has some doubts
on same topic which was asked earlier he/she can continue to ask her doubts on same thread.

In this application we have used following techstack:
-> Bootstrap
-> Express
-> Socket.io
